# WonderBear — 功能亮点

*AI 驱动的全球化儿童教育产品*

---

## 核心功能

### 1. AI 智能对话 — 和小熊聊天讲故事
3-8 岁孩子通过语音与 AI 小熊进行自然对话，AI 围绕孩子的想法构建个性化故事。对话轮次根据年龄自适应（3-5 岁 5 轮，6-8 岁 7 轮）。支持 **33 种语言** 的语音输入（Whisper ASR）和语音输出（DashScope CosyVoice + Azure Neural TTS 双引擎）。

### 2. AI 绘本自动生成
一次简短对话后，WonderBear 自动生成完整的 **10 页插画绘本**，每页配有双语语音旁白。封面图片由 Gemini 3 Pro 生成 2K 高清图，内页使用 FAL flux-pro 保持角色风格一致。整条管线全自动化，无需家长操作。

### 3. CLASS 系统课程 — 识字与阅读
体系化的识字阅读课程，对标权威教学标准：

- **中文课程**：500 个常用汉字，84 课——基于教育部人教版（PEP）小学 1-2 年级字表
- **英文课程**：200 个核心词汇，6 阶段 20 课——基于 Jolly Phonics 42 音素体系 + Dolch Sight Words
- **13 种语言课程**：波兰语、罗马尼亚语、俄语、西班牙语、德语、法语、意大利语、葡萄牙语、日语、韩语、泰语、越南语、阿拉伯语

每课完整学习闭环：**词汇学习 → 故事阅读 → 小测验**

### 4. 故事馆
浏览和发现 AI 生成的绘本。支持按语言和类别筛选，每页自动朗读。故事来自 AI 对话生成的个性化绘本和精心策划的公共藏书。

### 5. 收藏与下载
一键收藏喜欢的故事，跨设备同步。支持将整本绘本（含图片和音频）下载到本地，离线阅读。

### 6. 多子女档案
每台设备支持最多 4 个孩子档案，各自独立的学习进度、语言偏好和适龄内容（3-4 岁、5-6 岁、7-8 岁三档）。

### 7. 多语言界面
应用界面支持 **33 种语言** 实时切换。中文、英文、波兰语、罗马尼亚语四种核心语言拥有完整翻译。

### 8. 家长控制
扫码绑定设备、密码保护家长面板、内容安全双重过滤、使用管理和多设备家庭管理。

## 技术亮点

| 组件 | 技术方案 |
|------|---------|
| 大语言模型 | Gemini 2.5 Flash（对话 + 故事生成） |
| 图片生成 | Gemini 3 Pro → OpenAI → FAL（三级降级容错） |
| 语音识别 | DashScope Paraformer（中文）/ OpenAI Whisper（其他 32 种语言） |
| 语音合成 | DashScope CosyVoice / Azure Neural TTS（33 种语言） |
| 流式 TTS | Opus 编码低延迟流式管线 |
| 存储 | Cloudflare R2（图片：webp + HD PNG，音频：MP3） |
| 前端 | Vue 3 + TypeScript + Pinia |
| 后端 | Node.js + PostgreSQL + Redis |
| 支付 | Stripe（积分包 $1.50 起） |
| 实体书 | 按需印刷：32 页平装 $39.90 / 56 页精装 $59.90 |
